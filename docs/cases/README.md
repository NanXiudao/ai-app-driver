# Cases

This directory is for real examples and case studies.

Possible future contents:

- desktop app cases
- web app cases
- internal business system cases
- IDE or developer tool cases
- admin or content system cases

A useful case should ideally show:

- what the application looked like before
- how AI understood the codebase
- how features were extracted
- where the driver layer was placed
- what task skills were generated
- what worked
- what failed
