# Skills

This directory is for top-level work skills and task-oriented skills.

The current idea is to distinguish between two layers:

## 1. Top-level work skills

These describe how AI should perform major stages of application AI-enablement, such as:

- Codebase Comprehension
- Feature Inventory Extraction
- AI Driver Layer Design
- AI Driver Implementation
- Task Skill Authoring

These are not business functions.  
They are work-method skills.

## 2. Task skills

These are application-specific or task-specific `skill.md` files that tell AI how to complete concrete work inside an AI-driven application.

Examples:

- filter and export customers
- scan project status and generate a report
- batch update records and notify stakeholders

This directory will gradually host both kinds of skills and their structure.
